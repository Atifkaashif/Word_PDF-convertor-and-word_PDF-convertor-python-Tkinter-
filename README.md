# Word_PDF-convertor-and-word_PDF-convertor-python-Tkinter-
A simple Python Tkinter application to convert Word documents (.docx) to PDF and PDF files to Word (.docx) easily. It features a user-friendly GUI for quick file selection and seamless conversion

## Features
- Browse and select PDF files.
- Convert PDF text to Word format.
- Saves Word file in the same location as the PDF.
- User-friendly interface with Tkinter.

## Requirements
- Python 
- Libraries listed in `requirements.txt`

Install dependencies using:

```bash
pip install -r requirements.txt
